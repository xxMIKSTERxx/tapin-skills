# UI Kit — tapin.golf (marketing)

The public-facing marketing surface. Where TapIn explains itself to elite golfers, OEM partners, and the technical press.

This kit is **interpretive** — no codebase, Figma, or marketing copy was provided. It commits to the brand direction set in the root README:

- A dark hero. Imagery (computational putter visuals) carries the story.
- Sparse copy, numbers-led, no marketing hype.
- One product story per section. Whitespace is structural.

## Components in this kit

| File | Purpose |
|---|---|
| `index.html` | Single-page marketing site demo. |
| `components/SiteHeader.jsx` | Top nav: logo, links, fitting CTA. |
| `components/Hero.jsx` | "A putter, computed." + featured visual. |
| `components/Manifesto.jsx` | Three-statement positioning block. |
| `components/Pipeline.jsx` | How the CEM works — 4 numbered steps. |
| `components/SpecPanel.jsx` | Big numerical readout of a real Tap. |
| `components/SiteFooter.jsx` | Footer with secondary links. |
